Java Development Kit

Please make sure JDK is installed on your operating system.

To check your JDK version type :

javac -version in the terminal.

Depending on the version of Zookeeper, the latest JDK may beed to be installed.

Launching Zookeeper on Windows

Once zoo.cfg is ready, navigate to the bin folder using the command prompt, and run

  • zkServer.cmd

Note: No extra arguments are required


Launching Zookeeper on Linux or MacOS

Once zoo.cfg is ready, in your terminal, navigate to the bin/ directory and run

  • ./zkServer.sh start to start zookeeper

  • ./zkServer.sh stop to stop zookeeper


Asking for Help in the Q&A Section

To make the troubleshooting step fast and efficient, please follow these guidelines when asking for help in the Q&A section.

  1. Please make sure you read and followed all the above sections.

  2. Describe in detail your

    1. Operating System

    2. Exact Zookeeper file name downloaded.

    3. Version of JDK (see section on how to find out your JDK version)

    4. Contents of zoo.cfg

    5. The command you ran and the entire error message (including the stacktrace)

    6. Any troubleshooting steps you have already taken
